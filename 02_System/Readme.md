# CAPI System

This notebook contains the implementation of **CAPI**, a Gradio-based web application that enables users to interact with the chatbot through a web browser.

The system integrates a Large Language Model (LLM) with Retrieval-Augmented Generation (RAG) and Prompt Adaptation (PA) to answer questions related to the Informatics Study Program. The application was developed in **Google Colab** and uses **Gradio** to provide an interactive user interface for demonstration and user evaluation.

## Features

- Browser-based chatbot interface using Gradio.
- Retrieval-Augmented Generation (RAG) with configurable retrieval depth.
- Prompt Adaptation (PA) for grounding responses using retrieved knowledge.
- Semantic retrieval using Sentence Transformers.
- Automatic logging of user interactions, including queries, responses, timestamps, response time, and similarity scores.

## Running the System

1. Install the required Python libraries.
2. Configure the paths to the knowledge base and model files.
3. Execute all notebook cells sequentially.
4. Launch the Gradio application.
5. Open the generated local or public URL in a web browser to interact with CAPI.

## User Interaction

Users can submit free-text questions through the Gradio interface and receive responses generated by the configured LLM, RAG, and Prompt Adaptation pipeline. During each interaction, the system records conversation logs, timestamps, response time, and retrieval similarity scores for subsequent analysis.

## Reproducibility

This notebook reproduces the chatbot system used in the experiments reported in the accompanying paper. It enables researchers to deploy the same Gradio interface, evaluate different response-generation configurations, and collect interaction data for replication and further analysis.
