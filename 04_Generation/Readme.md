# Ablation Test

This notebook implements the ablation study conducted in this research to evaluate the contribution of each response-generation component in CAPI.

The experiments compare multiple system configurations by progressively enabling Retrieval-Augmented Generation (RAG), Prompt Adaptation (PA), and different retrieval depths. The objective is to quantify the impact of each component on response quality.

## Evaluated Configurations

The notebook evaluates the following response-generation configurations:

- **LLM-only**
- **LLM + RAG**
- **LLM + RAG + Prompt Adaptation (Top-1)**
- **LLM + RAG + Prompt Adaptation (Top-3)**
- **LLM + RAG + Prompt Adaptation (Top-5)**
